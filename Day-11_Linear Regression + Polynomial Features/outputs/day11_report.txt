KEY TAKEAWAYS
----------------------------------------------------------------------

1. Polynomial features capture nonlinear relationships in linear models
2. Degree 2 is usually the sweet spot — degree 3+ causes feature explosion
3. Ridge regularization is essential to prevent poly overfitting
4. RMSE penalizes large errors more than MAE
5. R² measures proportion of variance explained (1.0 = perfect)
6. Residual plots reveal systematic prediction errors
7. GPU neural net can capture arbitrary nonlinearities without manual features
